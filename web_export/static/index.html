    Column guide
    CAGR
    Compound annual growth rate: the average yearly return over the full available period, assuming gains are reinvested.
    Std Dev
    Standard deviation: a measure of how much returns moved up and down. Higher values usually mean a bumpier ride.
    Max Drawdown
    Maximum drawdown: the worst peak-to-trough fall in the TRI history.
    Sharpe
    Sharpe ratio: annualized average monthly excess return over monthly excess-return standard deviation, using 364-day T-Bill yields as the risk-free rate.
    Sortino
    Sortino ratio: annualized average monthly excess return over the standard deviation of negative monthly excess returns.
    Avg Rolling
    Average rolling return: the average of all rolling period returns, such as all 3-year or 5-year return windows.
    Worst Rolling
    Worst rolling return: the lowest return seen across the selected rolling windows.
    Best Rolling
    Best rolling return: the highest return seen across the selected rolling windows.
    Rolling Std Dev
    Standard deviation of rolling returns: a measure of how consistent the selected rolling-period returns were.
    Top Q
    Upper quartile count: how many rolling periods this index appeared in the upper 25 percent of the dataset.
    Top/Bottom
    Compares upper-quartile observations with lower-quartile observations.
    Bottom Q
    Lower quartile count: how many rolling periods this index appeared in the lower 25 percent of the dataset.
